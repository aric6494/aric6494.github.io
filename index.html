git端口22超时连接
发表于2025-02-02|问题
使用git push出现以下错误 1ssh: connect to host github.com port 22: Connection timed out 解决方法 在C:\Users\用户名\.ssh目录下新建config文件，用记事本打开该文件，粘贴以下内容 1234Host github.com Hostname ssh.github.com Port 443 User git 粘贴以下命令到命令行进行验证 1ssh -T git@github.com 然后应该会出现三个选项其中有yes/no/还有一个忘掉了，输入yes就可以了。成功后会出现以下内容 1Hi username! You've successfully authenticated, but GitHub does not provide shell access. 以上后半句话不影响我们使用git的相关操作， 只是不提供登录服务器的 Shell 终端权限。例如：你无法通过ssh git@github.com 登录到 GitHub 的服务器上执行 ls、cd 等命令。
